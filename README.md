# Kairos - Блокувальник сайтів з навчальним агентом

## Опис

Kairos — це програма, яка блокує доступ до сайтів, які ви визначите (наприклад, соціальні мережі). Коли ви намагаєтесь зайти на заблокований сайт:

1. Відображається мотиваційна цитата
2. Автоматично запускається агент
3. Агент аналізує ваш Google Calendar
4. Знаходить ваші поточні задачі з PDF-файлами
5. Генерує інтерактивний навчальний сайт на основі матеріалу

## Встановлення

### 1. Встановіть залежності

```bash
pip install -r requirements.txt
```

### 2. Налаштуйте Google API

1. Перейдіть на [Google Cloud Console](https://console.cloud.google.com/)
2. Створіть новий проект
3. Увімкніть Google Calendar API та Google Drive API
4. Створіть OAuth 2.0 credentials
5. Завантажте `credentials.json` і помістіть в папку проекту

### 3. Налаштуйте API ключ

Створіть файл `.env`:
```
ZAI_API_KEY=your_api_key_here
```

### 4. Запустіть mitmproxy

```bash
mitmproxy -s block.py
```

Або для тихого режиму:
```bash
mitmdump -s block.py
```

### 5. Налаштуйте проксі в браузері

Налаштуйте браузер на використання проксі:
- Хост: 127.0.0.1
- Порт: 8080

### 6. Встановіть SSL сертифікат

Перейдіть на `http://mitm.it` і встановіть сертифікат для вашого браузера.

## Використання

### Керування через веб-інтерфейс

Відкрийте `http://kairos.api` у браузері:

- **Додати сайт** — введіть URL і натисніть "Заблокувати"
- **Видалити сайт** — натисніть "Видалити" поруч із сайтом
- **Переглянути історію** — внизу сторінки показується історія блокувань
- **Налаштування** — натисніть на іконку налаштувань для підключення додатків

### Структура файлів

```
kairos/
├── block.py              # Головний файл mitmproxy
├── calendar_zai.py       # Агент для аналізу календаря
├── index-ai.html         # Веб-інтерфейс керування
├── style-ai.css          # Стилі веб-інтерфейсу
├── index.html            # Сторінка блокування (резервна)
├── blocked_sites.json    # Список заблокованих сайтів
├── block_history.json    # Історія блокувань
├── credentials.json      # Google OAuth credentials
├── token.json           # Google токен (створюється автоматично)
├── learning.html        # Згенерований навчальний сайт
└── .env                 # Змінні середовища
```

## Функції

### Автоматичний запуск агента

Коли ви намагаєтесь зайти на заблокований сайт:
1. Сайт блокується
2. Відображається цитата
3. Автоматично запускається `calendar_zai.py`
4. Агент створює `learning.html` з навчальним матеріалом

### Історія блокувань

- Зберігається останні 100 записів
- Показує сайт, час та цитату
- Автоматично оновлюється кожні 30 секунд

### Підтримувані додатки

- **Google Calendar** — читання подій з PDF-вкладеннями
- **Google Keep** — (планується)

## Вирішення проблем

### Сайти не блокуються
- Перевірте налаштування проксі в браузері
- Переконайтеся, що SSL сертифікат встановлено

### Агент не запускається
- Перевірте файл `.env` та API ключ
- Переконайтеся, що `credentials.json` існує
- Перший запуск потребує авторизації Google

### Помилка "No events found"
- Додайте подію в Google Calendar
- Прикріпіть PDF-файл до події
