📚 Основні поняття теорії ймовірностей

Експеримент та події

Експеримент - випробування, яке можна повторити за незмінних умов.

Випадкова подія - результат експерименту, який може статися чи не статися.

Приклад: Підкидання монети - експеримент. Випадання "герба" - випадкова подія.

Класичне означення ймовірності

P(A) = m / n

де m - число сприятливих результатів, n - загальне число рівноможливих результатів.

Типи подій

  • Достовірна - обов'язково станеться (P = 1)
  • Неможлива - ніколи не станеться (P = 0)
  • Випадкова - може статися чи ні (0 < P < 1)

Операції над подіями

Сума подій (A ∪ B)

Сталася хоча б одна з подій A або B

Добуток подій (A ∩ B)

Сталися обидві події A і B

Різниця подій (A \ B)

Сталася A, але не сталася B
